# Projeto Prático da Dio: Abstraindo um Bootcamp usando Orientação a Objetos em Java

Como parte do conteúdo avaliativo do Bootcamp BancoPan DIO - Digital Inovation One, abstraiu-se um projeto prático afim de implementar Programação Orientada a Objetos (POO) através de uma construção de um Bootcamp com a orientadora Camila Cavalcante.

O presente trabalho teve intençção de apresentar ao estudante uma visão geral de POO, implementando configurações do ambiente, conceitos como abstração e encapsulamento bem como herança e poliformismo na criação de classes do domínio tal qual sua interligação.

Dado posto ressalta-se que o projeto foi criado por Camila Cavalcante e que somente a utilização deste serviu como treinamento para futuros devs.
